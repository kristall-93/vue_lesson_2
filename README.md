# vue_lesson_2


Vue 3 + Vite

> Vite - https://vite.dev/guide/


## Установка

Клонируем проект:
```bash
git clone git@github.com:kristall-93/vue_lesson_2.git
```

Создание проекта:
```bash
npm create vite@latest
```

Переходим в директорию проекта:
```bash
cd vite-project
```

Устанавливаем зависимости:
```bash
npm install
```

## Запуск дев сервера
```bash
npm run dev
```
